# Portfolio Website

A responsive personal portfolio website built with HTML, CSS, and JavaScript.

## Overview

This project is a single-page portfolio site for showcasing a developer profile, services, projects, and contact information. It includes a navigation header, animated greeting text, responsive layout adjustments, and interactive UI elements.

## Features

- Responsive navigation menu for smaller screens
- Animated typing greeting in the hero section
- Sections for Home, About, Services, Projects, and Contact
- Project cards with placeholder buttons for future live links
- Custom dark theme with accent color and gradient buttons
- Smooth scroll behavior and hover effects

## Files

- `index.html` - Main website structure and content
- `style.css` - Styling and responsive layout rules
- `script.js` - Menu toggle and greeting text animation
- `Assets/Images/` - Project and profile images used by the site
- `Assets/Audios/` and `Assets/Videos/` - empty or reserved asset folders

## Usage

1. Open `index.html` in a web browser.
2. On smaller screens, tap the menu icon to toggle the navigation panel.
3. Use the nav links to jump to different sections of the page.

## Notes

- The contact form is currently a static form with no back-end submission logic.
- Project links are placeholders and can be updated to actual page URLs.

## Customization

To customize the site:

- Update text content and section headings in `index.html`
- Change colors, spacing, and layout in `style.css`
- Adjust greeting phrases in `script.js`
